BRD (Business Requirements Document)
Projet
Plateforme de Gestion Administrative et Pédagogique de l'Université Cheikh Hamidou Kane
Contexte
L'Université souhaite disposer d'une plateforme centralisée permettant la gestion des activités administratives, pédagogiques et du suivi des étudiants.
Actuellement, les informations sont dispersées entre plusieurs services, ce qui rend difficile :
la gestion documentaire ;
le suivi des étudiants ;
la gestion des formations ;
la communication institutionnelle ;
la production de statistiques.
L'objectif est de digitaliser ces processus au sein d'une application web unique.

Objectifs Métier
Objectif principal
Centraliser les opérations administratives et pédagogiques de l'université dans une plateforme unique.
Objectifs secondaires
Réduire les traitements manuels.
Faciliter l'accès aux documents.
Améliorer la communication interne.
Produire des statistiques fiables.
Assurer une traçabilité des actions.
Améliorer le suivi des étudiants.
